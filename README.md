# Orthodontic Payment Calculator

This is a simple web-based orthodontic payment calculator. It allows you to calculate monthly payments based on:

- Total Treatment Cost
- Insurance Cap (optional, max allowed by plan)
- Insurance Benefit
- Discount (flat amount or %)
- Down Payment
- Length of Payment Plan

## Usage

1. Open `index.html` in a browser to use locally.

2. Or upload this project to GitHub Pages to host online.

   - Create a new public repository

   - Upload `index.html` and `README.md`

   - Enable GitHub Pages in the repo settings

   - Your calculator will be live at `https://<username>.github.io/<repo-name>/`

## Printing

Click **Print Summary** to create a clean printout for patients.
